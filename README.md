# Quick Start API

## Installation

1. **Install Dependencies**

   ```bash
   bundle install

2. **Create Database**:
You need to have Postgres

    ```bash
    rails db:create
    ```

3. **Run Migrations**

    ```bash
    rails db:migrate
    ```

4. **Run Seeds**

    ```bash
    rails db:seed
    ```

5. **Run Server**

    ```bash
    rails s
    ```
